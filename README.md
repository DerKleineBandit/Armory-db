# ARMORY.db — Kanonen Datenbank

Kanonen-Datenbank mit Feuerraten-Simulator, Burst-Mechanik, Ordnern, Tags und Supabase-Backend.

## Setup

1. Supabase-Projekt anlegen auf supabase.com
2. SQL aus `supabase_schema.sql` im Supabase SQL-Editor ausführen
3. App öffnen → ⚙ Setup → URL + Anon Key eintragen

## Lokal starten

```bash
npm install
npm start
```

## Render Deploy

- Build Command: `npm install`
- Start Command: `node server.js`
